手撕包菜

发表于 2020-04-07 更新于 2020-04-28 分类于 湘菜

手撕包菜

食材：

包菜 1个
蒜 20克
干辣椒 一把
青花椒 一小把
盐 1.5克
糖 少许
味精 1克
油 20克
酱油 3克
香醋 1克

步骤：

包菜叶子手撕。
蒜切片，加干辣椒青花椒。
一勺底油，下蒜片辣椒青花椒。
下包菜爆炒不到一分钟。
调味加盐1.5g、味精1g、白糖少许。
加酱油3g、加少许香醋翻炒均匀出锅装盘。

葱油饼

发表于 2020-04-07 分类于 中式面点

葱油饼是一道家喻户晓的小吃，南北方都很常见，制作葱油饼的关键是起酥，今天国家一级面点师朱厘米就来教你做好葱油饼，制霸街头！

食材：

中筋面粉 100克
开水 50克
冷水10克
葱花 50克
盐 少许
油20克

步骤：

先用开水烫面，再加冷水，揉成面团。
下剂55克左右一个，搓成粗条静置10分钟。
案板刷油，粗条擀成长方形面片，表面刷油，撒盐，撒葱花，从宽处卷起成长条，接缝朝上，再从头部卷起，收尾处塞到底部，卷的少许松点。压扁后表面刷油静置。
静置10分钟后，擀圆，厚度在0.3厘米左右。
油锅温度在170度左右，煎至两面金黄即可。

发表于 2020-03-29 更新于 2020-04-23 分类于 技术

Flyway简介

Flyway是一个数据库版本控制管理工具。

###工作流程
Flyway将寻找他的迁移版本表中的元数据（flyway_schema_history），如果是空数据库则会创建他，用以跟踪数据库状态。
接着，扫描文件系统和应用类路径寻找sql或者java的迁移脚本，迁移将根据版本号排序并由历史表的metadata决定是否执行，脚本执行成功则会更新版本历史表。
