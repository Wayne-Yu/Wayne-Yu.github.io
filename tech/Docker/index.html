Docker

发表于 2020-08-07 更新于 2020-08-09 分类于 容器

简介

管理程序虚拟化（hypervisor vitualization）通过中间层将一台或多台独立的机器虚拟运行在物理硬件上，容器则是之间运行在操作系统内核上的用户空间。
容器虚拟化也称为”操作系统级”虚拟化，容器技术可以让多个独立的用户空间运行在同一台宿主机上，只能运行与底层宿主机相同或相似的操作系统。
